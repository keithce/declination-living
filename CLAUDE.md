# CLAUDE.md

Astrology app for planetary declinations and location matching. TanStack Start + Convex.

## Commands

- Dev: `bun --bun run dev` (frontend) + `npx convex dev` (backend)
- Build: `bun --bun run build`
- Test: `bun --bun run test`
- Lint/Format: `bun --bun run check`
